git commit --amend: thay đổi commit message
git commit --amend -m”<message>”: thay đổi commit message và thêm message mới
git restore --staged <file>: chuyển file từ vùng Stagging về vùng Working Directory
git reset HEAD~1: chuyển 1 file đã commit gần nhất từ vùng Repository về vùng Working Directory
